Linux目录文件显示统计

By Lake Villadom

发表于 2016-09-17

文章目录
  • 缘起：查看文件夹及其向下(递归)文件(夹)的信息。
[LV@LV-Cloud]$ ls -laR                   # -R, –recursive 同时列出所有子目录层
[LV@LV-Cloud]$ ls -l |grep "^-"|wc -l    # 统计某文件夹下文件的个数
[LV@LV-Cloud]$ ls -l |grep "^ｄ"|wc -l   # 统计某文件夹下目录的个数
[LV@LV-Cloud]$ ls -lR|grep "^-"|wc -l    # 统计文件夹（包括子文件夹）下文件的个数
[LV@LV-Cloud]$ ls -lR|grep "^d"|wc -l    # 统计文件夹（包括子文件夹）下目录的个数
