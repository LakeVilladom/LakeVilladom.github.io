关联特殊代码文件

VS Code 能识别大部分主流代码文件，但当碰到某些特殊文件如:tpl文件时，需要进行关联语法设置。

  • 配置：文件->首选项->工作区设置
        "files.associations": {
       "*.tpl": "html" 
    }
